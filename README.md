# TiskTask

это таск-трекер по типу шахматных часов. 
Вы выбираете задачу и начинаете работать над ней и запускается таймер. 
Если переключаетесь на другую задачу, то таймер стартует у другой задачи. 
И так каждое переключение. 
Ваша задача реализовать механизм переключения между задачами и чтобы сохранялось количество времени, потраченное на таску
